# EcoTrack - Carbon Footprint Tracker

This is a Next.js application built with Firebase Studio that allows users to track and reduce their carbon footprint.

## Key Features

*   **Activity-Based Data Collection**: Collects user inputs related to transport, electricity consumption, and food habits.
*   **Carbon Footprint Estimation**: Calculates weekly carbon emissions based on user data.
*   **Visual Footprint Breakdown**: Displays emissions distribution through interactive charts.
*   **Actionable Reduction Suggestions**: Provides personalized, practical steps each week to reduce emissions, powered by a GenAI model.
*   **Progress Tracking**: Monitors and compares carbon footprint trends over time.

## Getting Started

To get started, run the development server:

```bash
npm run dev
```

Then, open [http://localhost:9002](http://localhost:9002) in your browser to see the application.

- The main dashboard is at `/`.
- Log your weekly activities at `/log`.
- View your historical progress at `/progress`.
