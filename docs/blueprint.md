# **App Name**: EcoTrack

## Core Features:

- Data Collection: Collect user inputs related to transport, electricity consumption, and food habits.
- Footprint Calculation: Calculate weekly and monthly carbon emissions using standard emission factors, based on the data entered by the user.
- Emissions Breakdown: Visually represent emissions distribution through interactive charts and graphs.
- Personalized Tips: LLM tool that analyzes the user's carbon footprint data and generates 3-5 personalized, practical steps each week to reduce emissions, drawing upon a knowledge base of carbon-reduction strategies.
- Progress Monitoring: Track and compare carbon footprint trends over time to encourage sustainable behavior.

## Style Guidelines:

- Primary color: Forest Green (#388E3C) to represent nature and sustainability.
- Background color: Light Green (#E8F5E9), a very lightly saturated tint of the primary color.
- Accent color: Sky Blue (#64B5F6) to represent clean air and water; it complements the green while standing out enough to highlight important actions.
- Body and headline font: 'PT Sans', a humanist sans-serif.
- Use clean, modern icons to represent different categories of emissions and reduction tips.
- Prioritize a clean and intuitive layout for easy navigation.
- Use subtle animations and transitions to provide feedback and enhance user engagement.
